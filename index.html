Case Study — 提案事例

社内問い合わせ対応を自動化するAIチャットボットの提案
AIエージェント機能搭載チャットボット ― 業務課題から構成設計・デモ構築までを一貫して実施
1

想定課題

同じ内容の問い合わせが繰り返され、担当者の対応時間が圧迫されている。新入社員は必要な情報の所在が分からず、探索に多くの時間を費やしている。

2

提案内容

社内文書を検索し、要約して回答するAIチャットボットを導入。AIエージェント機能により、質問内容に応じて必要な情報源を自動的に判断し、適切な回答を生成します。

3

仕組み（非技術者向け）

社内に蓄積された文書データをAIが検索し、該当する情報を見つけ出して要約した回答を返します。これまで人が行っていた「探す・読む・まとめる」作業をAIが代行するイメージです。

4

期待効果

問い合わせ対応時間の削減、新人教育の効率化、属人化の防止、社内ナレッジの活用促進が期待できます。

5

実施内容（担当した役割）

課題の設定、システム構成の設計、チャットボットの作成、デモ環境の構築まで一貫して担当しました。

6

使用技術

Python OpenAI API LangChain チャットUI
GitHub リポジトリ デモ動画（1分）
